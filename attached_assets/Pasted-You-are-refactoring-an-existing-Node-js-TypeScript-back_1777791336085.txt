## INVARIANTS (CRITICAL)

Implement a function:

assertSessionInvariants(userId: string)

This MUST enforce:

1. Only ONE active session per user

   * active = endedAt is null

2. Session time validity

   * endedAt >= startedAt

3. No overlapping sessions for a user

If any invariant fails:

* throw a clear error
* DO NOT silently fix data

Call this invariant check AFTER:

* creating a session
* ending a session

endSession(userId, sessionId)
