Modify ensureSingleActiveTimer:

Add retry guard:

ensureSingleActiveTimer(userId, healFn, attempt = 0)

Behavior:

* If mismatch detected:
  IF attempt >= 2:
  - Log HARD ERROR:
  "self-heal failed after retries"
  - STOP (do not retry further)
  ELSE:
  - call healFn(userId)

This prevents infinite correction loops.
